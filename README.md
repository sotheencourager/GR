## Steps to Run the Cars.com Automated Test

1. Clone the repo - https://github.com/sotheencourager/GR.git
2. Download Visual Studio if you do not have it - https://visualstudio.microsoft.com/vs/community/
3. Open the cloned folder and navigate to the Solution and double click it - it should build it from there. If not, once the project is opened in VS, right click on the solution and select **Build Solution**.
4. If there are errors or warnings of missing packages, please right click on solution and select **Manage NuGet Packages for Solution**. Please ensure the packages pictures in the below are installed and updated.

![Packages](https://sowoyemi.com/wp-content/uploads/2021/03/packages.png)

5. Please navigate to the Test.cs file after all packages have been installed and the project is rebuilt. Please update line 16 of the cs file to reflect the folder path where the downloaded project is saved.
6. Click the Test menu from the menu bar and select Test Explorer from the sub menu
7. Click the Run All Tests (the green) button and the wait for the browser to be launched.

Please ask me any question you may have.
